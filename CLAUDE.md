## 작업 관리 (티켓)

이 프로젝트의 할 일은 `tickets/` 폴더에 **파일 1개 = 티켓 1개**로 관리한다 (2인 개발: dust9826 / D4LGONA).

- 세션 시작 / "뭐 할까" / 티켓 생성·수정·클레임 / 작업 분담 → **`maplerush-tickets` 스킬을 먼저 호출**하고 그 프로토콜을 따른다.
- 보드 보기: `node .claude/skills/maplerush-tickets/board.cjs`
- 현재 MVP 로드맵(테마1·1층 완주)은 `tickets/MR-*` 에 시드되어 있다 (`tickets/README.md` 요약).
